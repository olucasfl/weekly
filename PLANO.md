# Plano de construção — Rotina Weekly Planner

## Objetivo
Criar um app pessoal de organização semanal com backend Fastify e frontend React/Vite, separado em pastas e com design limpo inspirado em Claude.

## Fases concluídas
- Fundação: pastas api/, web/ e shared/
- Backend: Fastify, rotas de auth, categorias, tarefas, semana, metas e dashboard
- Frontend: tela de auth, semana, afazeres, eventos, progresso e perfil
- Qualidade: testes de recorrência, build do backend e frontend verificados

## Como rodar
- Backend: cd api && npm run dev
- Frontend: cd web && npm run dev
- API: http://localhost:3333
- Frontend: http://localhost:5173
